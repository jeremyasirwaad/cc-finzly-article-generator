## Wave 0 foundation (blocks everything)
- App shell / viewport → `AppShell` + `<router-outlet>`
- Material theme matching the Ext theme (palette + typography + `_extjs-parity.scss`)
- Auth/login + session sharing across both apps
- Global services: HttpClient interceptors (auth, error), error/snackbar handling
- Strangler bridge (reverse proxy) + route table generator

## Notes
- Composite pages (dashboards, anything embedding other pages) go in the **final** wave — they
  depend on their children being migrated first.
- High-reuse widgets are scheduled in Wave 2 so the shared-component extraction pays off downstream.
- This graph is living; re-run `/migration:plan` as the codebase is better understood.
